MIRemoteControl
===============

##简介

<br />
小米遥控器实现。通过监听Touch事件，来实现遥控器的简单功能。包括上下左右，菜单，主页以及返回等。

<br />
手势的即时显示通过贴图完成非绘制，而手势的判断则借助对点集合的简单分析。
